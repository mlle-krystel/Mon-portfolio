Résultats

J’ai transformé un portfolio statique en une véritable application web connectée à une API. Le site permet à l’architecte de gérer ses projets en autonomie via une interface d’administration sécurisée. Le code front est structuré avec des modules pour les appels API, la manipulation du DOM et la gestion des événements. Le résultat est une interface claire, dynamique et maintenable, pensée pour une utilisatrice non technique. Des pistes d’amélioration sont déjà identifiées (tests, accessibilité renforcée) pour aller encore plus loin.

Application fonctionnelle Interface modale ergonomique Gestion de contenu simple
Recul sur le projet

Ce projet m’a appris à raisonner en termes de parcours utilisateur et d’interactions, plutôt qu’en simple affichage de données. Avec le recul, je vois comment cette logique pourrait être approfondie, notamment en renforçant la gestion des erreurs et la robustesse des interactions dans un contexte de production réel.
