# Portfolio – Krystel Drappier

Portfolio d'intégratrice web réalisé dans le cadre de la formation
**Intégratur Web – OpenClassrooms**.

Ce site présente mon profil, mes compétences et plusieurs projets réalisés
durant mon parcours de formation.

---

## Objectifs

- Présenter mon profil et mon parcours de reconversion
- Mettre en valeur des projets concrets
- Appliquer les bonnes pratiques front-end
- Servir de support lors de la soutenance

---

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (ES6)
- React (composant isolé, sans build)

> Le site est majoritairement développé en HTML, CSS et JavaScript.  
> Un composant React a été intégré de manière ciblée afin de répondre  
> à la grille d’évaluation OpenClassrooms.

---

## Fonctionnalités

- Navigation fluide entre les sections
- Présentation de projets réalisés en formation
- Animations et interactions légères
- Carrousels de projets (Javascript et React)
- Design responsive

---

## Accessibilité & SEO

- Structure HTML sémantique
- Images avec attributs `alt
- Navigation clavier possible
- Balises `title` et `meta description`
